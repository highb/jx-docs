### Usage

```
jx gitops lint
```

### Synopsis

Lints the gitops files in the file system

### Examples

  ```bash
  # lint files
  jx-gitops lint --dir .

  ```

### Options

```
  -d, --dir string   the directory to recursively look for the *.yaml or *.yml files (default ".")
  -h, --help         help for lint
```
